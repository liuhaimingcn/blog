列出本地已有的镜像

  • REPOSITORY: 来自于哪个仓库
  • TAG: 镜像的标记（如果不指定具体的标记，则默认使用 latest 标记信息。）
  • IMAGE ID: 它的 ID 号（唯一）
  • CREATED: 创建时间
  • SIZE: 镜像大小
